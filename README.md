# Dgtle

swift4.2仿写数字尾巴，仅用于交流学习，侵删

### 技术点：
* 1、图文混排（文字表情）
* 2、图片浏览器（支持手势关闭、缩放）
* 3、banner
* 4、内存问题，原图很大（2000*2000），通过修改Kingfisher源码解决大图问题
* 5、瀑布流展示
* 6、详情页（H5展示，与原生交互等） TODO
* 7、不同类型的cell怎么优雅的展示（用到工厂模式、模板方法模式、适配器模式等）
